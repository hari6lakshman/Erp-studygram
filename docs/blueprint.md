# **App Name**: Classroom Central

## Core Features:

- Note Upload: Allow logged-in college users to upload class notes, specifying college, subject, and professor.
- Note Browsing: Enable users to browse available class notes by college, subject, and professor without login.
- Note Download: Provide a direct download link for each class note, accessible without login.
- User Authentication: Implement login functionality using predefined college usernames and passwords (stjosephs@123, iitm@123, saveetha@123, admin@123).
- Local Storage: Store notes and metadata to local database.

## Style Guidelines:

- Background color: Deep, rich blue (#1763B3), HSL(210, 100%, 43%)
- Card background color: Slightly darker blue (#125191), HSL(210, 100%, 36%).
- Primary/accent color: Vibrant gold (#D4A024), HSL(45, 84%, 49%) for borders, headers, and interactive elements.
- Body text/foreground: Soft, off-white (#D2D9E0), HSL(210, 20%, 85%) for readability.
- Headline font: 'Playfair', a modern serif with an elegant feel.
- Body text font: 'PT Sans', a humanist sans-serif, for all body text and chat messages.
- Central, card-like container with large rounded corners and a distinct gold border.
- Stylish horizontal gradient separator lines that fade from transparent to the gold accent color in the middle for header and footer.
